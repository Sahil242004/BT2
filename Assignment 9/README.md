# Assignment 9

## Problem
Deploy Public Key Infrastructure (PKI) based identity with infrastructure.

## What This Implementation Includes
- Simulated Certificate Authority (CA).
- Certificate issuance for users.
- Identity registry that verifies and stores trusted certificates.
- Message signing and verification using registered identity keys.

## Run

From inside this folder:

```bash
python3 pki_identity_demo.py
```

From workspace root (`Blockchain _assignments`):

```bash
python3 "Assignment 9/pki_identity_demo.py"
```

## Dependencies
No extra dependencies required (standard library only).

## Note
This is an educational PKI simulation using toy RSA values for demonstration.
