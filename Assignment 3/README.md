# Assignment 3

## Problem Coverage
Generate public/private keys and simulate a digital wallet.

## Run

From inside this folder:

```bash
python3 crypto_wallet.py
```

From workspace root (`Blockchain _assignments`):

```bash
python3 "Assignment 3/crypto_wallet.py"
```

## Dependencies
- Python package: `ecdsa`

Install only if missing:

```bash
python3 -m pip install ecdsa
```
