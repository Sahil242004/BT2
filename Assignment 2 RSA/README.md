# Assignment 2 RSA

## Problem Coverage
Generate RSA keys and perform basic message encryption/decryption flow.

## Build and Run

From inside this folder:

```bash
g++ -std=c++17 rsa.cpp -o rsa
./rsa
```

From workspace root (`Blockchain _assignments`):

```bash
g++ -std=c++17 "Assignment 2 RSA/rsa.cpp" -o "Assignment 2 RSA/rsa"
"./Assignment 2 RSA/rsa"
```

## Dependencies
No extra dependencies required (uses g++ only).
