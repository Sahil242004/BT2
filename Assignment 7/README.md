# Assignment 7

## Problem
Analyze Bitcoin block structure and compute Merkle root for sample transactions.

## Run

From inside this folder:

```bash
python3 bitcoin_merkle_analysis.py
```

From workspace root (`Blockchain _assignments`):

```bash
python3 "Assignment 7/bitcoin_merkle_analysis.py"
```

## Output Includes
- Basic block header field analysis.
- List of sample transaction IDs.
- Computed Merkle root using pairwise double-SHA256.
- Simplified block header hash for demonstration.

## Dependencies
No extra dependencies required.
