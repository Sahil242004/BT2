# Assignment 8

## Problem
Implement a simple Proof-of-Work mechanism for mining blocks.

## Run

From inside this folder:

```bash
python3 pow_mining_demo.py
```

From workspace root (`Blockchain _assignments`):

```bash
python3 "Assignment 8/pow_mining_demo.py"
```

## Output Includes
- Mining details for each new block (nonce, hash, mining time).
- Final chain validity check.

## Dependencies
No extra dependencies required.
